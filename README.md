# QuickCart 🛒

QuickCart is a modern Android grocery shopping application built using Kotlin and XML.

## Features

- OTP Login Authentication
- Persistent Login Session
- Grocery Product Listing
- Category Filtering
- Real-time Cart Updates
- Quantity Management
- Checkout Flow
- Order Success Screen
- Bottom Navigation
- Profile Management
- Responsive UI Design

## Tech Stack

- Kotlin
- Android SDK
- XML Layouts
- RecyclerView
- SharedPreferences
- ConstraintLayout

## Screens

- Splash Screen
- Login / OTP
- Home
- Categories
- Product Details
- Cart
- Checkout
- Profile

## Installation

### Clone Repository

```bash
git clone https://github.com/programmer-kunal/QuickCart.git
```

### Open Project

* Open Android Studio
* Click **Open**
* Select the cloned QuickCart project folder

### Build & Run

* Sync Gradle
* Connect Android device/emulator
* Click Run ▶️
